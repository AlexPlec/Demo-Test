# Demo Projects

Simple learning projects

## Technology list
- CS (EntityFramework, ASP.NET Core, SQLite)
- JavaScript (Node.js, SQLite, Vue.js)
- HTML
- CSS

## Prerequisites
- Node.js
- .NET 8.0
- SQLite

## Client

``` bash
cd toDo-list
npm install
npm run dev
```

## Server

``` bash
cd TodoListAPI
dotnet ef database update
dotnet run
```
